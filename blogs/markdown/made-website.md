Finally got around to making a personal website. Been meaning to do it for a while.
It exists now. That's the first step.

## Why

I wanted a place to write about things I'm building and thinking about. Twitter/X feels too noisy and stupid to me, a blog post sticks around and can actually be useful to someone searching for the same problem later.

Also, making the site itself was a fun little project, W Claude, couldn't have done this without you. Well maybe I could but definitely not in 2 days lmao. I wanted something that felt personal and not like a generic template so I chose this.

## The Stack

It's just static HTML, CSS, and a background image. No frameworks, no build step (well, except for this blog - I'm using Pandoc and a bash script for that).
The aesthetic is vaguely Windows 98/early web. I like how it looks and it's fast.
